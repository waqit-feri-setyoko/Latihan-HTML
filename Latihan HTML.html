Latihan Paragraf "p"

Tag HTML untuk membuat paragraf adalah Tag "P" tapi di HTML sendiri jika kita menuliskan sebuah paragraf bisa tidak menggunakan tag "p" jadi lansung bisa diketik saja dibawah tag "body".

latihan Bold "b", Underline "u", Italic "i"

tag HTML untuk membuat tulisan bercetak tebal menggunakan tag "b" atau "strong".

selanjut untuk meebuat tulisan bercetak miring bisa menggunakan tag "i" atau "em".

masih ada satulagi tag yait untuk membuat tulisan bergaris bawah bisa menggunkan tag "u".

Latihan Order List "ol" dan Unorder list "ul"

sekarang kita akan latihan membuat list berurutan atau order list dengan tag "ol" dan list tidak berurutan atau unorder list dengan tag "ul".

contoh list menggunakan order list.

  1. satu
  2. dua
  3. tiga

contoh list menggunakan unorder list.

  • bundar
  • kotak
  • lingkaran

Latihan mengubah style list (style="list-style-type:...")

contoh mengubah style di order list

  1. list menjadi angka (default) jika menggunakan list-style-type : "decimal"
  1. list menjadi angka yg didepannya ada angka nol"0" jika mengunakan list-style-type : "decimal-leading-zero"
  1. list menjadi romawi kecil jika menggunakan list-style-type : "lower-roman"
  1. list menjadi romawi besar jika menggunakan list-style-type : "upper-roman"
  1. list menjadi huruf abjad a-z kecil jika menggunakan list-style-type : "lower-alpha"
  1. list menjadi huruf abjad A-Z besar jika menggunakan list-style-type : "upper-alpha"

contoh mengubah list di unorder list

  • list menjadi lingkaran hitam(default) jika menggunakan list-style-type : "disc"
  • list menjadi lingkaran berwana putih jika menggunakan list-style-type : "circle"
  • list menjadi kotak hitam jika menggukan list-style-type : "square"
  • list menjadi tidak menggunakan apapun jika menggunakan list-style-type : "none"
